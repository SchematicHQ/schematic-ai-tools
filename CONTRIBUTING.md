# Contributing

Thanks for your interest in contributing to Schematic AI Tools!

## Getting Started

1. Fork the repository
2. Create a branch for your change
3. Make your changes
4. Open a pull request against `main`

## What to Contribute

- Bug fixes for incorrect SDK examples or types
- Updates to match new SDK versions
- Improvements to skill instructions or clarity
- New language/framework coverage

## Guidelines

- Keep examples concise and focused
- Test that code examples match the actual SDK APIs
- One logical change per PR

## Questions?

Open an issue if you have questions or want to discuss a change before starting work.
